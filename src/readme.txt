Building instructions
========================

1. Make sure that Python is installed (tested with Python 2.7)
2. Open the vcproj file, select "Distribution" configuration and build

Installation
=============

1. Copy pyhiew.hem to %HIEW%
2. Create directory %HIEW%\pyhiew and put the scripts there

This folder should contain:
- init.py (pyhiew initialization file)
- hiew.py (pyhiew python library)
- other scripts

Python packages
=================

You may need to install the Python for Windows extensions to be able to run some scripts:

- http://sourceforge.net/projects/pywin32/files/pywin32/Build%20214/pywin32-214.win32-py2.7.exe/download
